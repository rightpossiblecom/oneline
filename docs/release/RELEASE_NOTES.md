# OneLine Release Notes

## Version 1.0.0 - Initial Release

### What's New
- Launch of OneLine puzzle game
- 30 carefully crafted levels
- Progressive difficulty system
- Light and dark theme support
- Sound and haptic feedback controls
- Local progress saving
- Perfect path badge for clean solutions

### Features
- Minimalist design
- No ads or in-app purchases
- Privacy-first approach
- No data collection
- Offline gameplay

### How to Play
Touch the screen and draw a continuous line to connect all dots. Don't lift your finger until all dots are connected. Guide your line through increasingly complex patterns as you advance through levels.

### System Requirements
- Android 12.0 and above (or iOS 11.0)
- Minimum 2 MB free storage
- No internet connection required for gameplay

### Known Limitations
- None in v1.0.0

- Level editor (in planning)
